# Keyboard Maestro node shell script

## What is this

This is a small example of node shell script that takes input from stdin and slugifies it

```shell
npm install
```

## to run in the terminal

```shell
echo "b a n a n a" | ./slugify.js
```

or

```shell
npm test
```
